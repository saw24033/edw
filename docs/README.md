# Stepford County Railway Documentation

This directory contains documentation for various aspects of the Stepford County Railway system.

## Stations Documentation

Detailed documentation for individual stations can be found in the `stations/` directory.

### Available Station Documentation

- [Leighton West](stations/leighton_west.md) - Small terminus on the Leighton West Branch

## Adding New Station Documentation

When documenting a new station, please include:

1. **Overview** - Basic information, station code, type, operator
2. **Location** - District, zone, branch line
3. **Station Information** - Platforms, tracks, facilities
4. **Station Layout** - Platform-by-platform breakdown with services
5. **Services** - Routes serving the station
6. **Adjacent Stations** - Neighboring stations and distances
7. **Key Features** - Notable characteristics
8. **Historical Development** - Version history and major changes
9. **URL** - Link to wiki page

## Data Sources

Station information is sourced from:
- `scr_stations.json` - Complete station data extracted from the SCR Wiki
- SCR Unofficial Wiki: https://scr.fandom.com/
